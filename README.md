# Portfolio Data Analyst - AHMED SEFDINE

Application Streamlit pour présenter mon portfolio professionnel en tant que Data Analyst.

## Installation

1. Clonez le repository :
```bash
git clone portfolio-data-analyst
cd portfolio-data-analyst
```
2. Installez les dépendances :

```bash
pip install -r requirements.txt
```

3. Assurez-vous que votre API backend est en cours d'exécution :

```bash
cd portfolio-data-analyst
npm run dev
```

4. Lancez l'application Streamlit :

```bash
streamlit run main.py
```
